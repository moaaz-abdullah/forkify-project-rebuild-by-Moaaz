# 🍴 Forkify App

Forkify is a modern web application that allows users to search, view, bookmark, and upload recipes. The app is built using Vanilla JavaScript and follows the MVC (Model-View-Controller) architecture.

## 🚀 Features
- 🔍 Search for recipes using the Forkify API
- 📄 View detailed recipe information
- 🔖 Bookmark favorite recipes
- ➕ Upload your own custom recipes
- 🔄 Pagination for search results
- ⚡ Dynamic UI updates without page reload

## 🛠️ Technologies Used
- JavaScript (ES6+)
- MVC Architecture
- Parcel (for bundling)
- HTML5 & SCSS

- Working with APIs and asynchronous JavaScript (Promises, async/await)
- Implementing MVC design pattern
- State management and UI rendering

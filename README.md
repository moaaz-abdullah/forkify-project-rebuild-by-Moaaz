# 🍴 Forkify App
Forkify is a modern web application that allows users to **search, view, bookmark, and upload recipes**.
The app is built using **Vanilla JavaScript** and follows the **MVC (Model-View-Controller)** architecture.

---

## 🚀 Overview

Forkify provides a smooth and interactive cooking experience where users can:

* 🔍 Search for recipes from an external API
* 📖 View detailed recipe information (ingredients, cooking time, servings)
* 🔖 Bookmark favorite recipes
* ➕ Upload their own custom recipes
* 🔄 Adjust servings dynamically

---

## 🧠 Features

* Real-time recipe search
* Recipe details with dynamic servings adjustment
* Bookmark system (saved in local storage)
* Upload new recipes with validation
* Clean and responsive UI
* MVC architecture for better code organization

---

## 🛠️ Technologies Used

* JavaScript (ES6+)
* HTML5
* CSS3
* REST APIs (for fetching recipes)
* Local Storage

---

## 🏗️ Architecture

The project is structured using the **MVC pattern**:

* **Model** → Handles data and API requests
* **View** → Manages UI and rendering
* **Controller** → Connects model and view

## 🎮 How to Use

1. Search for a recipe using the search bar
2. Click on a recipe to view details
3. Adjust servings if needed
4. Bookmark recipes for later
5. Upload your own recipe using the form
